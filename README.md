# AppUsage-Statistics-Project 
[![Open In Colab](https://colab.research.google.com/assets/colab-badge.svg)](https://colab.research.google.com/github/AyaBestun/AppUsage-Statistics-Project/blob/main/Appusage%20Statistics.ipynb)

![image alt](https://github.com/AyaBestun/AppUsage-Statistics-Project/blob/c0dad9f6b80a53378a03e559e033ed0956cd91cc/app_usage%20logo.jpg)

A data driven exploration of mobile app usage patterns across 500 users, analyzing categories such as social media, education, and productivity to uncover behavioral insights using Python and data science tools.
---

## Dataset
The dataset contains 500 records with app usage across categories like social media, education, productivity, entertainment, health, news, and shopping — along with a target variable indicating high productivity.

---

## What This Notebook Covers


Part 1 – Descriptive Statistics
- Data exploration and summary statistics
- Histograms, box plots, count plots
- Correlation heatmap and pair plots
- Pivot tables and cross-tabulations

Part 2 – Basic Practice of Statistics
- Mean, median, variance, standard deviation
- Central Limit Theorem
- Z-scores and p-values
- T-tests, ANOVA, confidence intervals
- Pearson correlation
- Linear and logistic regression

Part 3 – Quartiles
- Quartile calculation and box plots
- Outlier detection using IQR
- Chebyshev's theorem
- Skewness visualization
